
Configurar proxy no SENAC para usar gitbash:

git config --global http.proxy USUARIO:SENHA@192.168.0.1:8080

exemplo (q funciona):
git config --global http.proxy 192.168.0.1:8080
_________________________________________________________________

Verificar o status do repositorio local:

git status

_________________________________________________________________

Clonar repositorio para a máquina local:

$ git clone https://github.com/kilaG/GpsInteligente.git

_________________________________________________________________

Adicionar alteraçoes:

git add --all

_________________________________________________________________

Fazer commit:

git commit -m "#4"

_________________________________________________________________

Configurar usuario local:

git config --global user.name "thaine"

git config --global user.email "tataabich@gmail.com"

_________________________________________________________________

Fazer o upload das alteraçoes:

git push

ou

git push origin master
